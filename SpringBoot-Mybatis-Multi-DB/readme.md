# Quick start

步骤 1 通过 docker composer 初始化 2 个 mysql

Step 1 Initialize multi mysql with docker composer

注：初始化脚本在 `mysql` 文件夹中 

PS: init script store in `mysql` directory

```
docker composer up .
```
